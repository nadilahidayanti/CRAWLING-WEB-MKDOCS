    Penjelasan Fungsi dari Library¶
    1. Sastrawi

    Steeming bertujuan untuk mentransofrmasikan kata menjadi kata dasarnya (root) dengan menghilangkan semua imbuhan kata (affixes) meliputi awalan kata (prefixes), sisipin kata (infixes), akhiran kata (suffixes) dan atau menghilangkan awalan dan akhiran kata (confixes) pada kata turunan begitulah fungsi dari Library Sastrawi yang digunakan untuk mengambil kata dasar dan membuang kata yang tidak penting

    1. BeautifulSoup4

    BeautifulSoup merupakan library yang terdapat di python untuk menarik data yang ada di HTML. ada banyak library yang dapat digunakan untuk menarik data pada web namun saya kali ini menggunakan BeautifulSoup.

    1. Sklearn

    Penyedia rumus yang digunakan untuk menghitung nilai K-Means dan Shilhouette

    1. Numpy

    Untuk mengolah data numerrik dan digunakan untuk operasi vektor dan matriks

    1. Request

    Digunakan saat me-Crawl data pada web .

    Note :¶

    Cara menjalankan library tersebut bisa menggunakan CMD disesuaikan dengan versi Idle Python nya atau jika tidak mendownload library satu persatu bisa menggunakan Aplikasi Spyder versi disesuaikan dengan laptop pengguna.
